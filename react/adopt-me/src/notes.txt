What are the two react script tags?
---

1. The first is the React library. This library is the interface of how to interact with
React; all the methods will be via this library. 

2. The second library is the rendering layer. In this case, since we're rendering to the browser, we're 
using react DOM.

2 types of react components:
---
1. function components - must return markup. 
2. class components
